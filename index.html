box-sizing

neko.css 将页面中所有元素的 box-sizing 置为 border-box， 这意味着元素的 width 和 height 属性将包括 padding 和 border， 详情 MDN

布局容器

使用布局容器来包含页面主体内容。

.neko-container 类占据100%的宽度，最大宽度980px。

.neko-container-fluid 类占据100%的宽度，没有最大宽度。

网格系统

.neko-row 类为网格系统的容器。自带一个 -8px 的 margin (gutters) 来抵消布局容器的边距。使用 .neko-no-gutters 类可以删除 gutters

.neko-row 类的子元素只能是 col，内容应该被放在 col 里，col 只能是 .neko-row 的子元素

neko 的网格系统使用 Flexbox 实现，要注意局限性和 Flexbox bugs。

每个 col 都有水平的 padding，你可以在 row 上添加 .neko-no-gutters 类来删除它

neko-col-*

.neko-col-* 类可以创建一个 col。.neko-col-* 其中的 * 代表该元素占据多少列，一行最多12列，超出的元素会换到下一行。

.neko-col-3
.neko-col-6
.neko-col-12

响应式网格

  • .neko-col-[1-12] 所有屏幕设备上都会生效，如手机、电脑等。> 0
  • .neko-col-sm-[1-12] 在超小屏幕及以上的设备上生效，如横着的手机。> 576px
  • .neko-col-md-[1-12] 在小屏幕及以上的设备上生效，如平板电脑。> 768px
  • .neko-col-lg-[1-12] 在中等屏幕及以上的设备上生效，如笔记本电脑。> 992px
  • .neko-col-xl-[1-12] 在更大屏幕设备上生效，如台式电脑。> 1200px

隐藏元素 neko-hidden

.neko-hidden {
  display: none; 
}

响应式隐藏 neko-hidden-*-[up/down]

  • .neko-hidden-*-up：在指定宽度及以上的设备上隐藏。
  • .neko-hidden-*-down：在指定宽度及以下的设备上隐藏。

边距工具

.neko-[m/p]-{$direction}-{$level}

  • m: margin
  • p: padding
  • $direction: l(eft), r(ight), t(op), b(ottom), h(orizontal), v(ertical)
  • $level: 1-5 之间的等级。每个等级之间相差 8px
